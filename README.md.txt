This project is a simple AI-based Customer Support Chatbot built using Python and Streamlit.
It helps users get quick answers to common customer queries.

## Features
- Handles order status queries
- Handles refund and cancellation requests
- Provides customer support contact details
- Fallback response for unknown queries
- Clean and simple web interface

## Tech Stack
- Python
- Streamlit

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py
